description(hover/press) ในส่วนนี้เป็นส่วนที่ถูกเพิ่มขึ้นมาภายหลัง จุดประสงค์เพื่อให้โปรแกรมสามารถรับค่าจากผู้ใช้เป็น text แล้วtext นั้นจะไปแสดงผลบนหน้าจอ โดยมีanimation เคลื่อนไหวอย่างเช่นตัวอย่าง

ทั้งนี้ปัญหาที่ผมพบเจอมีอยู่2กรณี

1.ฟังชั่น "splitting.js" ไม่สามารถทำงานได้ หากมีการรับค่าtextใหม่มาแทนที่เดิม ซึ่งเป็นค่าที่กำหนดไว้แล้ว splitting.js คืออะไร??

2.เพื่อให้ animation ทำงานได้อย่างเหมือนกันแม้ว่าตัวอักษรจะไม่เท่ากัน ผมจึงได้สร้างสมการในส่วนของCSSขึ้นมา ซึ่งใน1รอบลูป จะทำงานเป็นเวลา4000ms(4s) การจะให้เริ่มลูปใหม่ได้ทันพอดีกับตัวอักษรทั้งหมดจึงใช้สมการ [(--char-index + 1) x (4000ms/จำนวนตัวอักษร)] (ตัวแปร --char-indexจากฟังก์ชั่น splitting.js ของแต่ละตัวอักศรที่ถูกsplt เป็นตัวแปรที่อยู่ใน CSS) ซึ่งจำนวนตัวอักษรหาได้จากการกำหนดfunction ใน javascript (let text = document.getElementById("txtInsert").value; let length = text.length; )

ซึ่งปัญหาอยู่ในส่วนนี้ - การจะใช้ค่าจำนวนตัวอักษรนั้น เป็นตัวแปรที่อยู่ใน javascript ซึ่งตัวแปรใน js ไม่สามารถนำมาใช้ใน CSS ได้นอกจากเขียนคำสั่ง CSS ใน javascript แทน ซึ่งในระหว่างการเขียน ผมไม่สามารถเรียกค่าตัวแปรจาก CSS ให้มาเป็นตัวเลขได้ จึงต้องทำการศึกษาเพิ่มต่อไป - Thanut Promsupsin 29/12/2021
